# 📖 About the Project

HealthBase is an intelligent hospital emergency scheduling platform designed to demonstrate how **Operating System CPU Scheduling Algorithms** can be applied to solve real-world healthcare scheduling problems.

Unlike traditional hospital management systems that simply maintain patient records, HealthBase introduces an intelligent scheduling engine that allocates patients to doctors using classical CPU Scheduling techniques. The platform enables users to visualize scheduling execution, compare algorithm performance, and analyze important scheduling metrics through an interactive dashboard.

The project bridges the gap between theoretical Operating System concepts and practical software engineering by transforming CPU scheduling into a healthcare resource management solution.

# ⚙️ Scheduling Algorithms

HealthBase supports five major CPU Scheduling Algorithms:

### 🔹 First Come First Serve (FCFS)

Patients are treated strictly based on arrival time.

---

### 🔹 Shortest Job First (SJF)

Patients requiring the shortest treatment time are scheduled first.

---

### 🔹 Shortest Remaining Time First (SRTF)

The patient with the shortest remaining treatment duration receives priority.

---

### 🔹 Round Robin

Patients are executed in a cyclic manner using a fixed time quantum.

---

### 🔹 Priority Scheduling

Emergency patients receive higher priority over normal patients.

---

# 📊 Performance Metrics

The Scheduling Engine automatically calculates:

- Average Waiting Time
- Average Turnaround Time
- Average Response Time

- CPU Utilization

These metrics help compare the efficiency of different scheduling algorithms under varying workloads.
